✦ Ordering Prophecy — Topological Sort
For a DAG (no cycles), find a linear ordering where every edge goes forward. Uses Kahn's Algorithm: repeatedly remove nodes with in-degree 0, adding them to the order.
⏱ O(V+E)💾 O(V) Queue
